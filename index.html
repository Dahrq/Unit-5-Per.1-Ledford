Text Formatting

There are three main types of text formatting. Those are bold, italics, and underline.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An inline element does not display its content on a new line. It keeps the content on the same line as the previous content. Bold, italics and underline are all inline elements.

Attributes

Attributes are properties which can change an HTML element. Attributes are written in the opening tag of ANY HTML element. The most common attributes are src, href, class, id, and style.

Images

An image element is an inline element which inserts pictures into the document. An image element is self-closing, which means it does not have a closing tag. It has one major attribute, src, which is a filepath that points to the images/ location. Those filepaths can either be absolute or relative.

Image Attributes

There are 4 attributes commonly used for images. width and height are used to resize the images. alt allows the use of text description of images for accessibility and broken images. title allows a hover-over text to appear.

Width and Height

The width and height can be measured in many different ways, but the more common ones are pixels and percentages. Using pixels, you set the specific, precise size based on pixels. Using percentages, you set the dimensions based on a percentage of the entire page. In either case, setting the width alone respects aspect ratio, while setting both does not.

Alt Attribute

Every picture should at LEAST have the alt attribute. The alt attribute helps with screen readers for visually impaired and in the cases that the image is broken.
